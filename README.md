# TourBookingSystem

## Description
This project is a Tour Booking System developed for the CE302 Advanced Software Engineering course.

The system allows customers to browse tours, search and filter available options, and make bookings. Administrators can manage tours and view reservations.

## Features
- User login and registration
- Browse tours
- Search and filter tours
- Book tours
- Cancel bookings
- Admin management of tours

## Technologies Used
- Java
- JavaFX
- IntelliJ IDEA
